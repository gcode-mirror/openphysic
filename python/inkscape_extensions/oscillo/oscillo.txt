Oscillo Extension for Inkscape est une extension pour le logiciel de dessin vectoriel Inkscape


Fichiers :
	oscillo.txt : le présent fichier
	oscillo.inx : fichier XML permettant de faire apparaître le script dans le menu Extensions/Exemples/Oscillo
	oscillo.py : code source Python décrivant le tracé
